## Installation

### Option 1: One-click import
Click the "Import Blueprint" button above.

### Option 2: Manual import
1. Go to **Settings** → **Automations & Scenes** → **Blueprints**
2. Click **Import Blueprint**
3. Paste the raw URL of the blueprint

### Option 3: File copy
Copy the YAML file to your `config/blueprints/automation/` directory.

## Requirements

- Home Assistant 2022.5.0 or newer (for `not_from`/`not_to` trigger options)
